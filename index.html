HTML

O HTML significa Linguagem de Marcação de Hipertexto, é baseada em marcações que chamamos de tags. Uma tag é um conjunto de palavras entre sinais de colchetes angulares.
A maioria das tags possuem abertura e fechamento, o fechamento é identificado pela presença de uma barra.

H1~H6


Os elementos H1~H6 são elementos que representam níveis de títulos. Sendo o <h1> de maior tamanho diminuindo até o <h6>.

<p>


A tag <p> é utilizado para criar parágrafos.

<mark>

Representa um trecho de destaque de um texto "como um marca-texto", devido à sua relevância em um contexto.
O elemento <mark> é usado para destacar partes do texto.

<small>

Representa comentários com direitos autorais ou textos jurídicos, o texto fica em tamanho menor.

<i>

É utilizado para destacar parte do texto em "itálico" por algum motivo (normalmente por termos técnicos, expressões em outros idiomas).

<u>

Elemento do texto fica "sublinhado" nas versões antigas do HTML. Na versão do HTML5 deve incluir um estilo no CSS.

<strong>

O elemento é utilizado em contúdo de grande importância, como alertas. Diferente do <b> o <strong> dá significância para aquela parte específica do texto.

<ol>

O elemento <ol> representa uma lista ordenada, tem como característica uma lista mostrada como uma contagem, podendo ser numerais, letras, números romanos ou simbolos.
  1. primeiro item
  2. segundo item
  3. terceiro item

<ul>

O elemento <ul> representa uma lista desordenada, os itens não trazem uma ordenação numéricas e suas posições são irrelevantes.
  • primeiro item
  • segundo item
  • terceiro item

<li>

A Tag <li> é usada para representar um item que faz parte de uma lista. Este item deve estar contido dentro de um elemento pai, sendo ele <ol> ou <ul>.

<a>

O elemento é utilizado com o atributo href e cria uma hiperligação com as páginas web, arquivos, endereços de email, ligações na mesma página ou endereços URL.

<hr>

Faz aparecer uma linha horizontal.

<sub>

O texto em específico fica subscrito com uma linha de base reduzida, utilizado normalmente em expressões químicas.
C8H10N4O2

<sup>

O texto em específico fica sobrescrito em uma linha de base elevada, utilizado normalmente em expressões matemáticas.
a2 + b2 = c2

<blockquote>

O elemento indica que o texto é uma longa citação. A URL da fonte da citação pode ser dada usando o atributo cite e designa um documento de origem ou mensada para a informação citada.

<font>

A tag deixou de existir com a versão de HTML5, era utilizado como editor de fontess do texto. A formatação de font é feito através do CSS.
Era utilizado da seguinte forma: Aqui é seu texto que está com a fonte "ARIAL"

<del>

Representa uma parte do texto que doi excluída do documento.
Este texto foi excluído.

<abbr>

Representa uma abreviação e opcionalmente fornece uma descrição completa. Se presente o atributo tittle deve conter a descrição completa.
